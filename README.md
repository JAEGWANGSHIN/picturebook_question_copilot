# 그림책 질문수업 코파일럿

초등 교사가 그림책을 질문 중심수업으로 바꾸는 과정을 돕는 Streamlit 기반 MVP입니다. 교사는 수업 상황을 입력하거나 vFlat SCAN으로 만든 OCR 텍스트를 붙여넣어 그림책 추천, 질문 10개, 질문 중심수업 지도안, 학생 활동지, 짝 토론지, 마음성장노트형 성찰 질문지를 만들 수 있습니다.

## 1. 앱 소개

이 앱은 교사의 전문성을 대체하지 않습니다. 그림책을 아이의 마음을 여는 수업 매개로 보고, 질문 만들기와 짝 토론, 나만의 생각 쓰기를 중심으로 수업 설계를 돕는 보조 도구입니다.

핵심 흐름은 다음과 같습니다.

1. 수업 상황 입력
2. 그림책 추천
3. vFlat OCR 텍스트 자료화
4. 그림책 흐름에 따른 질문 10개 생성
5. 질문 중심수업 지도안 생성
6. 학생 질문 수업화
7. Markdown 또는 DOCX 내보내기

## 2. 주요 기능

### 상황 기반 그림책 추천

교사가 학년, 교과, 성취기준, 수업 주제, 학급 상황, 수업 시간, 원하는 활동 방식을 입력하면 내부 DB 기반 점수화로 그림책 3~5권을 추천합니다.

추천 점수 기준은 다음과 같습니다.

- 수업 주제와 main_themes 일치: 40점
- 학년 적합성: 20점
- 학급 상황과 emotion_keywords 일치: 20점
- 질문 생성 가능성: 10점
- 활동 확장성: 10점

### vFlat SCAN 자료화

PDF, TXT 파일은 텍스트 추출을 시도합니다. JPG/PNG는 MVP에서 자동 OCR을 하지 않으며, vFlat SCAN 앱에서 추출한 OCR 텍스트를 붙여넣는 방식을 기본으로 합니다.

생성 결과는 다음을 포함합니다.

- 그림책 내용 요약
- 주요 인물
- 핵심 사건
- 핵심 갈등
- 인물의 감정 흐름
- 수업 주제 후보
- 질문을 만들기 좋은 장면
- 수업 활동 아이디어
- 정서적 유의점

### 질문 10개 만들기

다음 흐름에 맞춰 질문 10개를 생성합니다.

1. 표지 또는 제목 질문
2. 그림 관찰 질문
3. 이야기 이해 질문
4. 인물 감정 질문
5. 행동 이유 질문
6. 관점 전환 질문
7. 가치 판단 질문
8. 삶 연결 질문
9. 공동체 실천 질문
10. 생각 쓰기 질문

각 질문에는 질문 유형, 질문 의도, 예상 학생 반응, 후속 질문, 정서 안전성 점검이 함께 제공됩니다.

### 질문 중심수업 지도안 만들기

40분 수업을 기본으로 하며, 교사가 수업 시간을 바꿀 수 있습니다. 도입, 전개 1, 전개 2, 전개 3, 정리 단계별로 시간, 교사 발문, 학생 활동, 예상 반응, 판서 또는 활동지, 교사 유의점을 표로 제공합니다.

### 학생 질문 수업화

학생 질문을 다음 유형으로 분류합니다.

- 사실 확인 질문
- 해석 질문
- 감정 질문
- 관점 전환 질문
- 가치 판단 질문
- 삶 연결 질문
- 공동체 실천 질문
- 위험하거나 사생활 노출 가능성이 있는 질문

수업화하기 좋은 질문 3개를 추천하고, 짝 토론 질문, 모둠 토론 질문, 글쓰기 질문, 활동지 질문으로 변환합니다. 위험하거나 사생활 노출 가능성이 있는 질문은 안전한 질문으로 바꾸어 제안합니다.

### 결과물 저장 및 내보내기

현재 세션에서 생성한 결과물을 선택하여 Markdown 또는 DOCX 파일로 다운로드할 수 있습니다.

## 3. 설치 방법

```bash
cd picturebook_question_copilot
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## 4. 실행 방법

```bash
streamlit run app.py
```

브라우저가 자동으로 열리지 않으면 터미널에 표시되는 Local URL로 접속합니다.

## 5. 환경변수 설정 방법

`.env.example` 파일을 복사해 `.env` 파일을 만듭니다.

```bash
cp .env.example .env
```

기본값은 mock mode입니다.

```env
LLM_PROVIDER=mock
OPENAI_API_KEY=
OPENAI_MODEL=gpt-4o-mini
LLM_TEMPERATURE=0.2
```

실제 LLM 연결을 사용하려면 다음처럼 설정합니다.

```env
LLM_PROVIDER=openai
OPENAI_API_KEY=여기에_API_키_입력
OPENAI_MODEL=gpt-4o-mini
LLM_TEMPERATURE=0.2
```

## 6. API 키 없이 mock mode로 실행하는 방법

아무 설정을 하지 않거나 `.env`에 아래처럼 둡니다.

```env
LLM_PROVIDER=mock
```

이 경우 외부 API를 호출하지 않습니다. 추천은 내부 DB 점수화로 작동하고, 질문 10개, 지도안, 학생 질문 수업화는 규칙 기반 샘플 생성기로 작동합니다. 강의 시연용으로 바로 사용할 수 있습니다.

## 7. vFlat SCAN 자료 사용 방법

1. vFlat SCAN 앱으로 그림책 일부 장면을 스캔합니다.
2. vFlat의 OCR 텍스트 추출 기능을 사용합니다.
3. 앱의 `vFlat SCAN 자료화` 탭에 OCR 텍스트를 붙여넣습니다.
4. 원문 전체를 복제하지 말고 수업 설계에 필요한 범위에서만 사용합니다.
5. 생성 결과는 요약, 질문, 활동 아이디어 형태로 사용합니다.

## 8. 저작권 유의사항

- 이 앱은 그림책 원문을 장문 재출력하거나 복제하기 위한 도구가 아닙니다.
- 스캔 자료는 수업 설계 보조용으로만 사용하세요.
- 그림책 전체 이미지 또는 원문 전체를 저장하거나 공유하지 마세요.
- 결과물은 원문 복제가 아니라 요약, 질문, 지도안, 활동지 형태로 변환됩니다.
- 업로드 파일은 앱 코드에서 장기 저장하지 않습니다. Streamlit 세션 안에서만 처리합니다.
- 실제 학교나 공개 강의에서 사용할 때는 원저작권자의 권리를 존중하고, 필요한 경우 정식 도서를 구입해 활용하세요.

## 9. 개인정보 보호 유의사항

- 학생 이름, 학교명, 반, 전화번호, 가족 정보 등 개인정보를 입력하지 마세요.
- 학생 질문 분석 기능에는 학생 질문을 익명 처리해 붙여넣으세요.
- 예시는 “학생 A”, “학생 B”처럼 처리하세요.
- 민감한 개인정보가 들어간 경우 분석 전에 반드시 제거하세요.

## 10. 정서 안전 유의사항

- 학생의 글이나 질문을 보고 심리 진단을 하지 않습니다.
- “이 학생은 불안장애가 있다”, “자존감이 낮다”처럼 단정하지 않습니다.
- “걱정, 외로움, 망설임 같은 감정 표현이 보입니다”처럼 관찰 중심으로 말합니다.
- 개인사, 가족사, 트라우마를 직접 묻는 질문을 피합니다.
- 개인 경험 질문은 항상 선택형 또는 간접형으로 제시합니다.
- 인물 중심 질문 → 나와 연결 질문 → 공동체 실천 질문 순서로 안전하게 확장합니다.

## 11. 테스트용 예시 입력

### 예시 1

- 학년: 초등학교 3학년
- 수업 주제: 다름을 존중하기
- 학급 상황: 친구의 외모나 말투를 놀리는 일이 있다.
- 원하는 활동: 짝 토론과 생각 쓰기

### 예시 2

- 학년: 초등학교 2학년
- 수업 주제: 속마음 표현하기
- 학급 상황: 친구에게 먼저 말을 걸지 못하는 학생들이 있다.
- 원하는 활동: 말 걸기 카드 만들기

### 예시 3

- 학년: 초등학교 5학년
- 수업 주제: 자존감과 자기수용
- 학급 상황: 친구의 평가에 예민한 학생들이 많다.
- 원하는 활동: 마음성장노트와 글쓰기

### 예시 4: 학생 질문 목록

```text
주인공은 왜 혼자 있었을까?
친구들이 놀리지 않았다면 어떻게 되었을까?
다르다는 것은 고쳐야 하는 걸까?
나도 친구에게 상처받은 적 있니?
우리 반에서 서로 다름을 인정하려면 무엇을 해야 할까?
```
